# pretty-vendor

基于Vue-CLI3.0，前端用Vue全家桶，后端用Koa2+Sequelize+Mysql，通过Nginx部署的一个SPA应用。


## 安装与运行

### 启动后端服务
 1. 进入server目录，安装依赖
 ```
  npm install
 ```
 2. server目录下启动koa2项目：
 ```
  npm nun dev
 ```
 

### 运行前端项目
1. 安装依赖

```
npm install
```

2. 运行项目
```
npm run serve
```

3. 打包构建
```
npm run build
```


## 历史版本

基于Vue-CLI2.0，前端请求使用Easy-mock，位于vue-cli2.0分支
